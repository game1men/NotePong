# NotePong
Pong but with Notepads

## Controlls
 - Player1:
   - Up: W  
   - Down: S 
 - Player2:
   - Up: ↑
   - Down: ↓ 
 - Game:
   - Ball Speed:
     - Up: P
     - Down: O
   - Player Speed:
     - Up: L 
     - Down: K  
   - Exit: Escape
